# Readme

这是一个用于总结论文、实时解释论文选段的工具，基于windows和wsl运行，通过python实现

> 提示：配合chrome的插件“划词翻译”使用效果更好

This is a tool that can be used to sum up and explain essay，base on windows and wsl，rely on python runtime. All the prompt are written in Chinese , but can be modified.

## 系统原理

核心功能通过两个python程序实现

* `psump.py`：输入一个**原生pdf**，调用大模型解析，得到一个md文档
* `pask`：输入一个`psump.py`生成的总结，依次作为模型对论文的基础语料，然后通过监听win上的剪切板，若内容发生改变，则会自动就其中的内容向大模型提问，得到回答

## System Introduce

The core functionality is implemented through two Python programs:

* `psump.py`: Takes a **raw PDF** as input, calls a large language model to parse it, and generates an `.md` document.
* `pask`: Takes a summary generated by `psump.py` as the model's baseline knowledge, then monitors the Windows clipboard. If the clipboard content changes, it automatically sends the new content to the large language model for questioning and receives answers.

## 提示词设置

在`config.json`中可以设置三个prompt:

* `system_prompt`：用于给llm提供身份
* `sum_prompt`：用于让其总结分析论文全文
* `ask_prompt`：用于对所选内容进行发文

## Prompt Settings

Three prompts can be configured in `config.json`:

* `system_prompt`: Used to provide the LLM with a role/identity
* `sum_prompt`: Used for summarizing and analyzing the full paper
* `ask_prompt`: Used for asking questions about the selected content

## 命令行接入

提供了三个bash文件（在`bash`文件夹下），将其放到`~/.local/bin`文件夹下，并赋予执行权限，直接调用命令`psum`/`pask`/`pagent`，但要改`psum`/`pask`中的python解释器和程序的路径

* `pagent`即是把两个功能合在一起，依次运行，并且提供了设置两个模型的命令行参数

## CLI Access

Three bash scripts are provided（under folder `bash`）. Place them in the `~/.local/bin` directory and grant execute permissions. You can then directly invoke the commands `psum`/`pask`/`pagent`, but you need to update the Python interpreter and program paths in `psum` and `pask`.

* `pagent` combines both features and runs them sequentially, with command-line arguments to set both models.

## Python程序的配置和使用

两个 Python 脚本用于处理学术论文：

| 脚本 | 功能 | 输入 |
|------|------|------|
| `psump.py` | PDF 论文总结 | PDF 文件 |
| `pask.py` | 论文提问助手 | 论文总结 MD 文件 |

#### 环境要求

- Python 3.10+
- WSL 环境（pask.py 需要调用 Windows PowerShell 访问剪贴板）
- Anthropic API 账号

#### 依赖安装

```bash
pip install pypdf anthropic rich
```

或使用项目提供的一键配置脚本：

```bash
./setup.sh
```

#### 模型配置

两个脚本共用模型配置，配置目录：`~/.anthropic_llm_profiles/`

每个模型是一个 `.sh` 文件，格式如下：

```bash
#!/bin/bash
export ANTHROPIC_AUTH_TOKEN="your-api-key-here"
export ANTHROPIC_BASE_URL="https://api.anthropic.com"
export ANTHROPIC_MODEL="claude-sonnet-4-20250514"
```

#### 添加新模型

创建配置文件，例如添加 deepseek 模型：

```bash
mkdir -p ~/.anthropic_llm_profiles
cat > ~/.anthropic_llm_profiles/deepseek.sh << 'EOF'
#!/bin/bash
export ANTHROPIC_AUTH_TOKEN="your-api-key"
export ANTHROPIC_BASE_URL="https://api.anthropic.com"
export ANTHROPIC_MODEL="claude-sonnet-4-20250514"
EOF
chmod +x ~/.anthropic_llm_profiles/deepseek.sh
```

#### 使用方法

**1. psump.py - PDF 总结**

```bash
# 列出可用模型
python psump.py model list

# 设置默认模型
python psump.py model set deepseek

# 处理 PDF 文件
python psump.py run paper.pdf              # 输出到 paper 总结.md
python psump.py run paper.pdf -o out.md   # 指定输出文件
```

**2. pask.py - 论文提问**

```bash
# 设置提问模型
python pask.py model set deepseek

# 启动剪贴板监听
python pask.py run "论文标题 总结分析.md"
```

运行时：
- 在 Windows 中复制任意文本，自动发送给 AI 分析
- 输入 `p` 暂停监听，`r` 恢复
- `Ctrl+C` 退出

#### 配置文件

`config.json` 包含以下配置项：

| 字段 | 说明 |
|------|------|
| `system_prompt` | 系统提示词（角色设定） |
| `sum_prompt` | 总结提示词 |
| `sum_model` | PDF 总结使用的模型 |
| `ask_model` | 提问使用的模型 |
| `ask_prompt` | 提问时的 prompt 模板 |

#### 一键配置

运行 `setup.sh` 自动完成：
1. 创建 Python 虚拟环境
2. 安装依赖
3. 创建模型配置目录
4. 创建示例模型配置
